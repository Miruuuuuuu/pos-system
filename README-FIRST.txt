To run and test your POS system, follow these 3 simple steps:

┌──────────────────────────────────────────────────────────────┐
│ STEP 1: Install Frontend Dependencies                        │
└──────────────────────────────────────────────────────────────┘

   Double-click: install.bat
   
   (This will install all required packages for the frontend)


┌──────────────────────────────────────────────────────────────┐
│ STEP 2: Make Sure MongoDB is Running                         │
└──────────────────────────────────────────────────────────────┘

   Open Command Prompt and type: mongod
   
   Keep that window open while using the app!


┌──────────────────────────────────────────────────────────────┐
│ STEP 3: Start the Application                                │
└──────────────────────────────────────────────────────────────┘

   First time: Double-click seed.bat (adds sample data)
   Then: Double-click start.bat (starts the app)


┌──────────────────────────────────────────────────────────────┐
│ 🌐 ACCESS THE APPLICATION                                     │
└──────────────────────────────────────────────────────────────┘

   Open your browser to: http://localhost:3000
   
   Login with:
   📧 Email: admin@pos.com
   🔑 Password: admin123
